# My Web Page - Learning Web Development

This repository contains a collection of HTML, CSS, and JavaScript files that serve as exercises and experiments for learning web development fundamentals.

## Project Structure

The project is organized into various HTML files, each focusing on a specific web technology or concept:

- **Introduction & Basics**: 
  - `introduction to .html`: Basic HTML tags, headings, and paragraph structure.
  - `class and id.html`: Introduction to styling with classes and IDs.
  - `formating of.html`: Text formatting examples.

- **Intermediate HTML**:
  - `tableand list in .html`: Working with HTML tables and lists (ordered/unordered).
  - `HTMLform.html`: Building web forms for user input.
  - `iframe and file path .html`: Using iframes and understanding file paths.

- **Graphics & Multimedia**:
  - `HTmL canvas.html`: Basic 2D graphics using the HTML5 Canvas API.
  - `scrable vector graphix.html`: Scalable Vector Graphics (SVG) experiments.
  - `11.mp4`, `BAD GUY.mp3`, `gh.png`, `duo part 4.png`: Media assets used across the project.

- **JavaScript**:
  - `Javascript.html`: Basic JS implementation and alert functions.
  - `my.js`: External JavaScript file for logic.
  - `html Event.html`: Handling user interactions via DOM events.

## Getting Started

To view any of these exercises, simply open the desired `.html` file in any modern web browser.

1. Clone or download this repository.
2. Navigate to the folder.
3. Double-click on any `.html` file (e.g., `introduction to .html`) to open it in your browser.
